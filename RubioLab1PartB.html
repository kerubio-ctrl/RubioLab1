DOCTYPE statement
- A DOCTYPE statement is placed at the beginning of an HTML document with the purpose of informing the web browser specifically about the document type.
How: OL, UL, DL?
- An ordered list starts with the ol tag and each list item starts with the li tag
- An unordered list starts with the ul tag and each list items starts with the li tag
- The dl tag defines the description list, the dt tagdefines the term (name), and the dd tag describes each term
Text editor vs Word Processor
- Text editor is best used when writing code, creating scripts, or any task where the file needs to be pure, unformatted text
- Word Processor is best used when creating documents that require specific formatting, visual appeal, professional presentation, or collaborative editing
What is the client-server model
- The client-server model is a network architecture where a client sends requests for services or resources, and a server provides responses or performs tasks
FTP
- File Transfer Protocol (FTP) is a standard network protocol for transferring files between computers on a TCP/IP network
- Operates on a client-server model with separate control and data connections
HTTP
- Hypertext Transfer Protocol (HTTP) is the foundational network for the World Wide Web
- Defines how web browsers and servers format and transmit data to display web pages and other content
TCP/IP
- Transmission Control/Internet Protocol (TCP/IP) is the fundamental suite of protocols that governs how data is exchanged across the internet and other networks, breaking data into packets, addressing them, routing them to their destination, and reassembling them accurately
